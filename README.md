****Vehicle Rental Management System ****

Live URL: https://assignment2-one-snowy.vercel.app/

**Key Features**

*User & Auth Module*  
- User registration and login  
- Role-based access control (Admin & Customer)  
- Secure authentication using JWT

*Vehicle Module*  
- View all vehicles  
- Track vehicle availability  
- Automatic status update when booked or returned  

*Booking Module*
- Create bookings with start and end dates  
- Automatic total price calculation  
- Customer can cancel booking before start date  
- Admin can mark booking as returned  
- Vehicle availability updates automatically based on booking status  

**Technology Stack**
  
*Backend*  
- Node.js  
- Express.js  
- TypeScript  
- PostgreSQL  
- JWT Authentication  

*Database*  
- PostgreSQL  

*Deployment*  
- Vercel

**Setup & Usage Instructions**.
- Install dependencies
- Create a .env file
- Access the application(http://localhost:3000)
